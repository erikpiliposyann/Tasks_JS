# 🚀 JavaScript Practice Tasks

## 📖 Overview

This project contains 21 JavaScript exercises designed to improve fundamental programming skills and strengthen understanding of JavaScript data types, functions, strings, arrays, and validation techniques.

## 🎯 Topics Covered

### 🔤 Strings

* Count characters in a string
* Convert text to uppercase
* Reverse a string without built-in methods
* Check whether a substring exists

### 📚 Arrays

* Find the index of an element
* Calculate the sum of array elements
* Iterate through arrays using loops

### 🔢 Numbers

* Add two numbers
* Check whether a number is even or odd
* Validate numbers using:

  * `typeof`
  * `Number.isNaN()`
  * `Number.isFinite()`
  * `Number.isSafeInteger()`

### 🧩 Type Checking

* Determine the exact type of a value
* Detect plain objects
* Detect primitive values
* Correctly handle:

  * `null`
  * Arrays
  * Functions
  * `NaN`

### ✅ Boolean Logic

* Check whether a value is truthy or falsy
* Convert values explicitly to booleans

### ⚖️ Equality Comparisons

* Compare values using:

  * Loose equality (`==`)
  * Strict equality (`===`)

## 🎓 Learning Goals

By completing these exercises, you will:

✅ Understand JavaScript data types better
✅ Learn to validate and process user input
✅ Work confidently with strings and arrays
✅ Master truthy and falsy values
✅ Understand the difference between `==` and `===`
✅ Handle common JavaScript edge cases
✅ Write clean and reusable utility functions

## 🛠️ Prerequisites

* Variables
* Functions
* Conditionals (`if`)
* Loops
* Arrays
* Objects
* Strings
* Basic JavaScript syntax

## 💡 Purpose

These exercises focus on building a strong JavaScript foundation before moving on to advanced topics such as Objects, Prototypes, Closures, DOM Manipulation, and Asynchronous JavaScript.

Happy Coding! 🎉💻
